# Financial Performance Analysis Using Python

## Project Overview
This project demonstrates a complete financial data analysis workflow using Python, Pandas, NumPy, Matplotlib, and Seaborn.  

The analysis was developed for a small investment company managing an equity fund consisting of approximately 150 U.S.-based companies across multiple industries and states.

The primary objective of this project was to analyze company financial performance, identify potential financial risks, and support data-driven portfolio rebalancing decisions through structured programming and customized visualizations.

The project was completed across three major phases:

- Program Planning & Algorithm Design  
- Python-Based Data Analysis Implementation  
- Data Visualization & Reporting  

---

# Business Problem
Investment firms regularly evaluate financial performance indicators to make informed portfolio management decisions. However, manually analyzing large financial datasets can be inefficient, error-prone, and difficult to scale.

The organization required a Python-based analytical workflow capable of:

- Importing and organizing financial data
- Detecting duplicate records
- Performing grouped descriptive statistical analysis
- Identifying financially risky businesses
- Calculating financial ratios
- Generating clear visual insights for stakeholders

The solution provides an automated and structured financial analysis process using Python programming and business intelligence techniques.

# Dataset Description
Dataset used in this project:

- `D598 Data Set.xlsx`

The dataset contains financial information for approximately 150 U.S.-based companies.

### Key Variables

| Column | Description |
|--------|-------------|
| `Business ID` | Unique identifier for each company |
| `Business State` | State where the company operates |
| `Total Long-term Debt` | Long-term business debt |
| `Total Equity` | Company equity value |
| `Debt to Equity` | Debt-to-equity financial ratio |
| `Total Liabilities` | Total liabilities of the business |
| `Total Revenue` | Company revenue |
| `Profit Margin` | Business profit margin |

# Python Analysis Features

## Duplicate Detection
The program identifies duplicate records to maintain data integrity and analytical accuracy.

## Descriptive Statistics
Grouped statistics were generated by business state using:
- Mean
- Median
- Minimum
- Maximum

## Financial Risk Analysis
Businesses with negative Debt-to-Equity ratios were filtered to identify potential financial concerns.

## Debt-to-Income Ratio Calculation
A new calculated metric was created:

```python
Debt-to-Income = Total Long-term Debt / Total Revenue
```

The analysis also safely handled divide-by-zero scenarios using conditional logic and `NaN` values.
